# LuxeHost - Site Web de Conciergerie Airbnb

Ce site web présente les services de LuxeHost, une conciergerie de luxe pour Airbnb à Fès, Maroc.

## ✨ Fonctionnalités

- **Design professionnel** : Interface moderne avec animations et effets visuels
- **Icônes vectorielles** : Utilisation de Lucide icons pour une apparence cohérente
- **Responsive design** : Optimisé pour tous les appareils
- **Animations fluides** : Transitions et effets hover sophistiqués
- **Formulaire de contact** : Formulaire complet avec validation
- **Section témoignages** : Avis clients pour renforcer la confiance

## 🏗️ Structure du projet

```
taha/
├── assets/
│   ├── index-c6g_3qqv.css    # Styles Tailwind CSS
│   └── custom.css            # Animations et styles personnalisés
├── index.html                # Page d'accueil
├── tarif.html                # Page tarifs détaillés
├── contact.html              # Page contact avec formulaire
└── README.md
```

## 🎨 Technologies utilisées

- **HTML5** : Structure sémantique
- **Tailwind CSS** : Framework CSS utilitaire
- **Lucide Icons** : Bibliothèque d'icônes vectorielles
- **Google Fonts (Inter)** : Typographie moderne
- **CSS personnalisé** : Animations et effets avancés

## 📱 Pages

### Accueil (index.html)
- Hero section avec image de fond et call-to-action

- Témoignages clients

- Tableau comparatif des services

### Contact (contact.html)

- Informations de contact

## 🚀 Déploiement

Ouvrez `index.html` dans un navigateur web moderne pour voir le site. Toutes les ressources sont locales, aucun serveur requis.
